# Tutorial: Portfólio do GitHub para Estudantes de TI

## Introdução

Ter um portfólio no GitHub é uma das melhores maneiras de mostrar suas habilidades na área de Tecnologia da Informação. Ele funciona como uma vitrine dos seus projetos, estudos e conhecimentos para recrutadores, empresas e outros desenvolvedores.

Neste tutorial, você aprenderá como criar e organizar um portfólio profissional no GitHub.

---

# 1. Criando sua Conta no GitHub

1. Acesse o site oficial do GitHub.
2. Clique em **Sign up**.
3. Escolha:
   - Nome de usuário
   - E-mail
   - Senha
4. Confirme sua conta.

## Dica

Escolha um nome profissional, pois ele fará parte do link do seu perfil.

### Exemplo

```bash
https://github.com/seuusuario
```

---

# 2. Configurando o Perfil

Após criar a conta:

1. Vá em **Settings**
2. Adicione:
   - Foto de perfil
   - Nome
   - Bio
   - Localização
   - Links profissionais

## Exemplo de Bio

```markdown
Estudante de TI | Desenvolvedor Front-end | Apaixonado por tecnologia e programação.
```

---

# 3. Criando o README do Perfil

O README do perfil é sua apresentação principal.

## Como criar

1. Crie um novo repositório
2. O nome deve ser exatamente igual ao seu usuário
3. Marque:
   - ✅ Public
   - ✅ Add a README file

---

# 4. Estrutura Básica do README

## Exemplo

```markdown
# Olá, eu sou João 👋

## Sobre mim
Estudante de Tecnologia da Informação apaixonado por desenvolvimento de software.

## Tecnologias
- HTML
- CSS
- JavaScript
- Python

## Contato
- LinkedIn
- E-mail
```

---

# 5. Adicionando Projetos ao Portfólio

Crie repositórios organizados para seus projetos.

## Exemplos de projetos

- Calculadora
- Sistema de Login
- API em Python
- Site Responsivo
- Banco de Dados SQL

## Estrutura recomendada

```bash
📁 nome-do-projeto
 ┣ 📄 README.md
 ┣ 📂 src
 ┣ 📂 assets
 ┗ 📄 LICENSE
```

---

# 6. Escrevendo um Bom README para Projetos

Todo projeto deve ter:

## Título

```markdown
# Sistema de Cadastro
```

## Descrição

Explique o objetivo do projeto.

## Tecnologias usadas

```markdown
- HTML
- CSS
- JavaScript
```

## Como executar

```bash
git clone URL
```

---

# 7. Fixando Projetos no Perfil

1. Vá ao perfil
2. Clique em **Customize your pins**
3. Escolha os melhores projetos

## Dica

Fixe:
- Projetos completos
- Estudos importantes
- Trabalhos acadêmicos
- Projetos pessoais

---

# 8. Boas Práticas

## Organize seus repositórios

Use nomes claros:

```bash
sistema-login
portfolio-html-css
api-python
```

## Faça commits organizados

```bash
feat: adiciona tela de login
fix: corrige erro de autenticação
docs: atualiza README
```
